## Install

Apps required (skip if already installed in your workstation)

1. NodeJS
2. Git Bash
3. Hyper
4. Python
5. [Visual Studio Build Tools](https://github.com/nodejs/node-gyp#on-windows)

6. on root folder run:

```sh
yarn bootstrap
```

7. On VSCode. install the following extensions:
   - Prettier
   - ESLint

## Usage

```sh
yarn start
```
